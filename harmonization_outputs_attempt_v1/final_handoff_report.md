## 3. Fields still intentionally unresolved

These remain `NA` in the harmonized output because no source
documentation supports a deterministic derivation. They are
flagged in `flagged_for_review.csv` for human input.

### bor_bin (all 9 template trials)
- Template does NOT track BOR.binary alone (SD splits 99/178 between bor_bin=0/1).
- Hypothesis: `1 iff (BOR.binary='R') OR (BOR.binary='SD' AND pfs_time>=183)` — match 86% / 78% on s1400i / EAY131_Z1D.
- **NOT committed** per the no-silent-guessing policy.

### pfs_bin (all 9 template trials)
- Template does NOT track pfs_stat directly.
- Hypothesis: `1 iff (pfs_time>=183) OR (pfs_stat=0)` — match 81% / 81% on s1400i / EAY131_Z1D.
- **NOT committed** per policy.

### S1400I `age`
- Template age values are decimals (per-sample). Source `age_num` is integer enrollment age.
- Per-sample collection date NOT present in any S1400I source file.
- **Will stay NA** unless an external sample-collection-date file is supplied.

### `Cimac.id` for 10013 and 14C0059G (new trials)
- Exhaustive scan of every cell of every file in both trial dirs for any string starting with a known `cimac_part_id` and followed by a sample-suffix: **zero hits**.
- External CIMAC sample manifest required.
